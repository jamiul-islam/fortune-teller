# Product Overview

iTellFortune is a fortune-telling consultation booking platform that provides an immersive, mystical user experience with seamless payment processing.

## Core User Flows

1. **Discovery**: Users land on an immersive landing page with fullscreen video backgrounds, smooth scroll animations, and consultation category information
2. **Booking**: Users book consultations through Calendly inline embed with integrated payment processing (Calendly handles all payment via Stripe)
3. **Confirmation**: After payment, users receive a unique confirmation token and access personalized content including welcome video, fortune notes, and solution cards

## Key Features

- Fullscreen video backgrounds with fallback images
- Smooth scroll-based animations using Motion.js
- Calendly Premium inline embed for booking
- Webhook-based booking confirmation
- Token-based access to personalized post-payment content
- Random fortune note selection from curated collection
- Mobile-responsive design with mystical premium aesthetic

## Business Model

- Fortune teller receives instant alerts via Calendly mobile app and SMS for immediate client callback
- Payment processing handled entirely by Calendly (Stripe integration)
- Consultation durations: 15, 30, or 60 minutes
- Token-based confirmation expires after 7 days
